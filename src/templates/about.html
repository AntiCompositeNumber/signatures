{%trans%}About{%endtrans%}

{%trans%}What is this?{%endtrans%}

{%trans%} This tool, called Signatures, is designed to find technical issues in custom user signatures. It was written in Python by User:AntiCompositeNumber. {%endtrans%}

{% trans %}Why does this exist?{% endtrans %}

{% trans trimmed proposal=''|safe, close=""|safe %} MediaWiki has never imposed technical restrictions on fancy signatures other than maximum length. This allowed editors to create complex signatures that are hard for machines to parse. Complex signatures sometimes contained mistakes that would affect other text. A {{proposal}}proposal{{close}} was created for technical requirements that would fix these issues. {% endtrans %}

{% trans trimmed %} This tool was created as a result of that proposal so that editors discussing the proposal could make informed comments. It may also be useful in notifying users about their problematic signatures. {% endtrans %}

{% trans %}What problems are detected?{% endtrans %}

{% trans trimmed %} As this tool was built to collect data about the proposed new requirements, it checks for signatures that would violate those requirements. {% endtrans %}

{% trans %}The following problems are detected:{%endtrans%}

  • {% trans trimmed lintopen=''|safe, close=""|safe %} Any {{lintopen}}Lint errors{{close}} {% endtrans %}
  • {% trans trimmed %} Missing or invalid links to user, user talk, and contributions pages, including signatures with {% endtrans %}
    • {% trans %}No user links at all{% endtrans %}
    • {% trans trimmed %} Only links to the user, user talk, or contributions page of someone else {% endtrans %}
    • {% trans %}Only links to a user page on another wiki{% endtrans %}
  • {% trans %}Fancy signatures without any formatting{% endtrans %}
  • {% trans trimmed %} Signatures longer than 255 characters (before substitution) {% endtrans %}
  • {% trans %}Signatures that change when the page is saved again{% endtrans %}

{% trans %}My signature works fine, why did the tool find problems?{% endtrans %}

{% trans trimmed %} Some lint errors reported by the tool may not currently affect the rendered output of the page. Some are deprecated tags that may not be supported by browsers in the future, while others are silently fixed by the wikitext rendering system. There is no way to guarentee that these lint errors will continue to work over time. {% endtrans %}

{% trans trimmed %} Other reported errors are a result of requirements that don't target the rendered version of a page. These requirements are designed to make it easier for machines to find and parse signatures. If these problems are not fixed, tools, scripts, and bots may not work correctly with your signature. {% endtrans %}

{% trans trimmed talkopen=''|safe, ghopen=''|safe, close=""|safe %} Some reported issues are false positives caused by bugs in the detection code. You should report these bugs to AntiCompositeNumber on {{talkopen}}their talk page{{close}} or on {{ghopen}}GitHub{{close}}. {% endtrans %}
